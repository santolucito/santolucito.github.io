Server Files

Put these files on some other machine. Then send a stream of messages to the server. For example, use the HyperIMU app. Any app that streams sensor value to a user selected IP and port is fine.

Since the server listens to your phone on one port, and sends events out another port, we cannot use this code with a globally accesible web server (ie running on port 80, or on port 443 for SSL). Instead, you can run the server wherever you like, but the client needs to be flexible and not demand a particular port (e.g. being hosted a https site
